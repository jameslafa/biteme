## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/jameslafa/biteme.git
   cd biteme
   ```

2. Start a local server:
   ```bash
   python -m http.server 8000
   ```

3. Open your browser to `http://localhost:8000`
